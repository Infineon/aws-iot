◆ THRESHOLD_YIELD_TIMEOUT

#define THRESHOLD_YIELD_TIMEOUT   1000

Threshold yield timeout (in ms).

Recommended threshold timeout value is 1000ms in order to allow adequate time for the system to receive and decode data. Recommended Yield timeout has been arrived at by considering different networks having different speed.
