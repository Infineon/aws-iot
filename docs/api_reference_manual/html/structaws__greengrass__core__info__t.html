aws_greengrass_core_info_t Struct Reference
AWS IoT library » AWS IoT and Greengrass data structures

Description

Greengrass Core Information retrieved from AWS.

A device can be part of multiple groups and can select to which Group's core it wants to connect to. Note: Each group have only one core.
